# 🛒 Blinkit Sales Data Analysis using Python

## 📌 Project Overview
This project analyzes Blinkit's sales data to uncover insights about product performance, customer behavior, and outlet trends using Python.

---

## 🎯 Business Objective
To identify key patterns in sales and provide data-driven insights for better decision-making.

---

## 📊 Key KPIs
- 💰 Total Sales  
- 📈 Average Sales  
- 📦 Number of Items  
- ⭐ Average Rating  

---
